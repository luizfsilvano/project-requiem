# Assets/_Project

Tudo que for específico do jogo deve nascer aqui.

Use esta pasta para evitar misturar código e assets próprios com arquivos padrão, samples, pacotes externos e configurações internas da Unity.

## Pastas

- `Art`: modelos, texturas, sprites auxiliares e assets visuais próprios.
- `Audio`: sons e música.
- `Materials`: materiais usados pelo projeto.
- `Prefabs`: objetos reutilizáveis.
- `Scenes`: cenas principais e cenas de teste nomeadas.
- `Scripts`: C# do jogo.
- `Settings`: configurações próprias do projeto.
