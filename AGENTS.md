# Orientações para agentes

- Leia a documentação do projeto antes de implementar.
- Não crie sistemas antecipadamente.
- Não derive a arquitetura definitiva dos templates `ThirdPerson` ou `Variant_Combat`.
- Mantenha conteúdo próprio dentro de `/Game/ProjectRequiem`.
- Prefira C++ para bases estáveis e contratos; use Blueprint para composição e apresentação.
- Não crie múltiplos módulos ou plugins próprios sem necessidade concreta.
- Não implemente gameplay quando a tarefa solicitar apenas fundação.
- Siga as convenções documentadas em `PROJECT_REQUIEM_ARCHITECTURE.md`.
- Utilize Conventional Commits.
- Nunca adicione arquivos gerados da Unreal ao Git.
